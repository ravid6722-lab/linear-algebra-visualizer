# Step 20.51 — Change of Basis Visual Clarity

This update improves the **Change of Basis** visualization so the core idea is clearer in both **2D** and **3D**.

## What changed

### 2D
- Kept the standard axes visible.
- Added highlighted **basis axes** for `u` and `v`.
- Added the scaled vectors **`αu`** and **`βv`**.
- Added the translated copy edges that complete the parallelogram.
- Added a clearer result vector **`w`**.
- Kept a faint basis-cell parallelogram and added a stronger coordinate parallelogram for `αu + βv`.
- Expanded adaptive zoom so the new basis-decomposition vectors stay visible.

### 3D
- Kept the standard `x,y,z` axes visible.
- Kept the basis plane visible with `u` and `v`.
- Added **`αu`**, **`βv`**, the translated copy edges, the coordinate parallelogram, and the result vector **`w`**.
- Reduced clutter by fading the raw basis vectors slightly while emphasizing the coordinate decomposition.

### Live Insight
- Improved the explanation text for **Change of Basis**.
- Added explicit readouts for:
  - `B = {u, v}`
  - `w`
  - `[w]_B = (α, β)`
  - `αu`
  - `βv`
  - `w = αu + βv`

## Goal of the fix
The visualization now better emphasizes that the important idea is:

- `u` and `v` define the new basis,
- `α` and `β` are the coordinates of `w` in that basis,
- and `w` is built geometrically as the diagonal of the parallelogram generated by `αu` and `βv`.
