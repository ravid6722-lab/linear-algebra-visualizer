# עדכון אחרון — בסיסים, מטריצות מעבר והנפשה

הגרסה כוללת את סבב התיקונים האחרון של המרצה.

## בדיקה מהירה

1. הפעילו את `frontend` ואת `backend` לפי `README.md`.
2. היכנסו לתרגול עצמאי ובחרו **בסיס, קואורדינטות ומעבר בסיס** בתצוגת 2D.
3. במצב **הזנת שני בסיסים**, שנו את איברי `P_B` ו־`P_C`.
4. במצב **הזנת מטריצת מעבר**, שנו את `T(C←B)` וראו כיצד בסיס C נגזר ומשתנה בגרף.
5. שנו את `[w]B`, `[w]std` או `[w]C` ובדקו שאותו וקטור מתעדכן בכל הייצוגים.
6. גררו את `b₁`, `b₂`, `c₁`, `c₂` ואת `w` ישירות על הגרף.
7. עברו לטרנספורמציה ליניארית וסמנו **הצג את המישור כצירופים ליניאריים של עמודות A**.
8. הפעילו את ההנפשה מהסרגל שמעל הגרף. המטריצה נשארת מוצגת באופן קבוע והמישור נע.

## בדיקות שבוצעו

- בניית Production של Vite הושלמה בהצלחה.
- נבדקה המרה אלגברית הלוך וחזור בין שני בסיסים.
- כל קובצי השרת עברו בדיקת תחביר.
- קובצי `package-lock.json` מפנים אל מאגר npm הציבורי.
