Future Full App · Roadmap Planned

This standalone HTML file is the demo phase. The full implementation will expand it into a real classroom platform. Items below are intentionally out of scope for the demo.

🔄
Real-time classroom sync
Socket.io broadcasts every lecturer action to connected student screens, with presence and reconnection handling.
🧊
Advanced 3D visualizations
Basic 3D is live in this demo. Future work: 3D eigen-axis visualization, animated camera fly-throughs, and richer lighting/shading.
🧮
n×n matrices & abstract spaces
Beyond 2D/3D: arbitrary-dimensional matrices, abstract vector spaces, and symbolic computation via Math.js — SVD, full eigendecomposition.
📤
Export lesson snapshot
Save a frozen state of the canvas + matrix + explanation as a PDF or PNG, so lecturers can reuse it on slides.
⚛️
React + Node.js stack
Componentized React frontend, Node.js/Express backend, persistent classroom rooms with lecturer-issued join codes.
✅
Numerical correctness tests
Automated tests validating every calculation (det, inverse, eigenvalues) against Python/NumPy or MATLAB reference outputs.
